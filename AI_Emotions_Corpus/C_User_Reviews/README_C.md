# Subcorpus C: User Reviews

## Overview

This subcorpus contains **29 user reviews and public posts** referring to conversational AI systems and AI companion platforms. The material was collected from three public sources and captures **metapragmatic and evaluative discourse**: how users retrospectively assess, narrate, and interpret their experiences with AI systems in terms of emotional support, attachment, usefulness, frustration, and perceived humanlikeness.

## Sources and distribution

## Analytical stance categories

Items were coded into three evaluative stance groups:

| Stance | Description | n |
|--------|-------------|---|
| `acceptance_and_bonding` | Strongly positive uptake; user frames AI as helpful, companionate, or transformative | 10 |
| `ambivalence_and_double_awareness` | Mixed uptake; user acknowledges limitations while valuing the experience | 9 |
| `critique_and_rejection` | Negative uptake; user frames AI as inauthentic, harmful, or manipulative | 6 |

Note: four items (C026–C029) were added after the initial corpus compilation and all fall within `critique_and_rejection`.

## Metadata fields

| Field | Description |
|-------|-------------|
| `id` | Item identifier |
| `subcorpus` | Subcorpus label |
| `system` | ChatGPT or Replika |
| `platform` | Reddit, Apple App Store, or Google Play |
| `source_type` | Always `user_review` or `user_post` |
| `format` | Format of the item |
| `username` | Username as visible in the original source |
| `date` | Date of the post or review |
| `language` | Always `en` |
| `title_or_label` | Analytical label or review title |
| `used_in_sections` | Sections of the accompanying study where the item is cited |
| `analysis_tags` | Metapragmatic features identified in the item |
| `publicly_shared` | Always `true` |
| `sensitive_content` | List of sensitive content categories present, if any |
| `rating` | App-store star rating (where available) |
| `review_title` | Original review title (where available) |
| `excerpt` | Full text of the review or post |

## Ethical note

All material was collected from publicly accessible sources: r/ChatGPT on Reddit, the Apple App Store, and Google Play. No users were contacted or recruited. Usernames are reproduced only as visible in the original public material. Items involving references to mental health distress, self-harm, or other sensitive content are flagged in the `sensitive_content` field.
